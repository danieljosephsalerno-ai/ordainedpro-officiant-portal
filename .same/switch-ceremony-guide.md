# Switch Ceremony Feature - User Guide

## Overview
The **Switch Ceremony** feature allows you to manage multiple wedding couples and toggle between them seamlessly. All ceremony data, wedding details, and communication are saved per couple.

## How to Use

### 1. Access the Switch Ceremony Button
- Located in the **top right header** of the portal
- Next to the "Active Ceremony" badge
- Click the **"Switch Ceremony"** button with the Users icon

### 2. View All Couples
When you click the button, a dialog opens showing:
- **All couples** you're managing
- Each couple's **bride and groom information**:
  - Names and avatars
  - Email addresses
  - Phone numbers
- **Wedding details**:
  - Venue name and location
  - Wedding date and time
  - Expected number of guests

### 3. Switch Between Couples
- The **currently active** couple is highlighted with:
  - Blue border
  - "Currently Active" badge
- To switch to a different couple:
  - Simply **click on their card**
  - The portal will immediately update to show that couple's information

### 4. What Gets Updated
When you switch couples, the portal updates:
- Wedding Couple section (bride and groom info)
- Wedding Details (venue, date, time, guests)
- All saved messages and conversations
- All saved scripts and files
- All tasks and meetings
- All contracts and payments

## Adding New Couples
To add a new couple/ceremony:
1. Click the **"Add New Ceremony"** button in the header
2. Fill in all the ceremony and couple details (at minimum: Ceremony Name, Bride Name, Groom Name)
3. Click "Add Ceremony"
4. **The new couple is automatically:**
   - Added to the Switch Ceremony dialog
   - Set as your currently active couple
   - Ready for immediate use
5. You can switch between all your couples anytime using the Switch Ceremony button

## Data Persistence
- All data is automatically saved per couple
- When you switch between couples, their specific data is loaded
- No data is lost when switching
- Each couple maintains their own:
  - Ceremony scripts
  - Files and documents
  - Messages and conversation history
  - Tasks and meetings
  - Wedding details and preferences

## Tips
- Use the Switch Ceremony feature to manage multiple weddings efficiently
- All changes are automatically saved for each couple
- The currently active couple's information is always displayed
- You can switch between couples as many times as needed
